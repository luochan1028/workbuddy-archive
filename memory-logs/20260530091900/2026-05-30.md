## 2026-05-30 早盘涨停定时任务

- 用户要求创建定时任务：抓取A股早上10分钟内涨停的票，中午通过微信推送，并监控成交量
- 编写了 `early_zt_alert.py` 脚本，核心逻辑：
  1. 通过 AKShare stock_zt_pool_em 获取当日涨停数据
  2. 筛选首次封板时间 <= 094000 的票（竞价涨停092500、开盘秒板0925-0930、5分钟板0930-0935、10分钟板0935-0940）
  3. 通过 stock_zh_a_hist 获取近5日成交量均值，标记放量>1.5倍的票
  4. 生成HTML表格报告（含涨停类型、封板时间、炸板次数、连板数、成交额、换手率、封板资金、成交量信号）
  5. 通过 PushPlus 推送微信
- 今日测试：17只早盘涨停票，推送成功
- 定时任务配置：每个交易日（周一至周五）12:00 执行，automation id=10
- 数据文件：early_zt_YYYYMMDD.csv
