# 全球投资情报看板 — 产品需求文档 v2.0

## 用户画像

- **身份**：中国A股投资者，专注"打板"策略，使用7层产业链+5维评分框架
- **核心诉求**：通过监控全球关键人物推文，获取投资方向信号 + 理解世界格局变化
- **使用场景**：晚间复盘为主，长期投资参考
- **被监控账号**：@realDonaldTrump, @elonmusk, @CathieDWood, @justinsuntron, @federalreserve

## 模块一：每日复盘简报（首页核心）

### 功能描述
打开看板的第一屏，5分钟读完今日全球关键情报。

### 内容结构
1. **今日格局一句话总结**（AI生成）
   - 例："今日特朗普对中国关税语气明显缓和，美联储鸽派信号增强，马斯克力挺AI开源，整体对A股偏暖"
   
2. **关键信号 Top 5**（按投资影响排序）
   - 每条信号标注：来源账号 → 信号类别 → 影响板块 → 方向（利好/利空/中性）→ 置信度
   - 例：⚠️ 特朗普提"重新评估对华关税" → 关税信号 → 出口贸易板块 → 潜在利好 → 中高置信度

3. **板块影响一览表**
   - 今日被提及的A股板块，标注利好/利空方向与提及次数
   
4. **风险预警**
   - 自动提取所有包含制裁/冲突/禁止/战争等关键词的推文
   
5. **跨账号共振话题**
   - 当2个以上账号24小时内讨论同一主题时，高亮聚合
   - 标注"共识度"和方向一致性

### 交互
- 每条信号可点击展开看到原始推文
- 支持切换日期查看历史某一天的简报

## 模块二：板块热度图

### 功能描述
可视化展示今日所有推文对A股各板块的影响分布。

### A股板块分类（10个）
| 板块 | 映射关键词 |
|------|-----------|
| 半导体/芯片 | chip, semiconductor, TSMC, 芯片, 制裁, export control |
| 新能源/电动车 | EV, Tesla, battery, solar, 新能源, electric vehicle |
| AI/大模型 | AI, LLM, GPT, Grok, xAI, artificial intelligence |
| 出口贸易 | tariff, trade war, export, 关税, 贸易 |
| 数字货币 | Bitcoin, crypto, blockchain, BTC, ETH, Web3 |
| 金融/利率 | Fed, rate, interest, inflation, 利率, 通胀 |
| 地缘/军工 | war, conflict, military, defense, 地缘, NATO |
| 消费电子 | Apple, iPhone, supply chain, 供应链 |
| 能源 | oil, energy, OPEC, gas, 能源 |
| 医药 | pharma, biotech, FDA, 医药 |

### 可视化方式
- **热力矩阵**：行=板块，列=账号，颜色深度=信号强度
- **红绿配色**（中国股市惯例）：红=利好，绿=利空
- 点击任意格子展开该账号在该板块的所有相关推文

## 模块三：格局追踪器

### 功能描述
不是看单条推文，而是看"趋势"——同一个大人物对同一个话题的态度变化。

### 追踪维度
1. **中美关系温度计**
   - 追踪特朗普对华言论的"硬度"变化（最近30天趋势线）
   - 对比马斯克对华态度（他是中国市场的晴雨表）
   
2. **美联储政策路径**
   - 从美联储推文/声明中提取利率信号
   - 标注：鹰派/鸽派/中性
   
3. **加密货币政策风向**
   - 特朗普+马斯克+孙宇晨三人对Crypto的言论汇总
   - 信号一致性分析（三人同时看好/看空？）
   
4. **AI科技竞争态势**
   - 马斯克xAI + 木头姐ARK + 孙宇晨Web3 综合信号

### 可视化
- 时间轴 + 信号强度柱状图
- 每个节点可点击查看当时的具体推文

## 模块四：推文详情库

### 功能描述
可搜索、可筛选的推文档案库。不是简单列表，每条推文都是"情报卡片"。

### 每条推文展示
```
┌─────────────────────────────────────────┐
│ 🔴 高优先级  │ 特朗普 · 2小时前         │
├─────────────────────────────────────────┤
│ 🇬🇧 "We need to reconsider the tariff   │
│      policy with China..."             │
├─────────────────────────────────────────┤
│ 🇨🇳 "我们需要重新评估对华关税政策…"      │
├─────────────────────────────────────────┤
│ 📊 投资解读：                            │
│ 特朗普罕见使用"reconsider"一词，相对于   │
│ 此前的"punish""confront"，语气明显软化和  │
│ 务实。这可能预示着中美贸易谈判窗口重新打开。│
├─────────────────────────────────────────┤
│ 🏷️ 信号标签：关税/贸易  利好出口板块      │
│ 📈 影响板块：出口贸易 ↑                   │
│ 💡 置信度：中高（75%）                    │
└─────────────────────────────────────────┘
```

### 与旧版的关键区别
- 加了一条 **"投资解读"**（不是直译，是专业解读推文对A股意味着什么）
- 每张卡片有明确的 **信号标签** 和 **板块标注**
- 优先按 **信号优先级** 排序，不是时间排序

### 筛选/搜索
- 按账号筛选
- 按信号类别筛选（关税/利率/AI/加密/地缘…）
- 按板块筛选
- 按日期范围
- 全文搜索（中英文）
- 按优先级筛选（只看高优先级信号）

## 模块五：账号画像

### 功能描述
每个被监控账号的"情报档案"，帮助理解这个人在你的投资框架中的角色。

### 内容
1. **基本信息**：头像、简介、关注理由
2. **关注领域标签**（自动统计最近30天发言主题分布）
3. **发推频率**（日/周活跃度曲线）
4. **对中国态度指数**（仅对特朗普、马斯克有意义）
   - 基于历史推文自动评估，标注趋势（变硬/变软/持平）
5. **近期关键言论**（Top 5 高信号推文）
6. **历史高光时刻**（之前发过的对市场有重大影响的推文）

## 模块六：数据&配置

### 账号管理
- 查看当前5个账号
- 一键添加/删除（预留功能，不常变）

### 信号规则管理
- 查看当前信号分类和关键词
- 支持自定义添加关键词（比如你想加"稀土"相关监控）

### 数据导出
- 导出今日/本周/自定义日期范围的推文为CSV
- 导出格式包含：原文+翻译+解读+信号标签+板块

## 数据库升级计划

### tweets 表新增字段
- `signal_tags` TEXT — 信号标签（逗号分隔，如"关税/贸易,中国市场"）
- `sectors` TEXT — 影响板块（逗号分隔）
- `impact_direction` TEXT — "positive"/"negative"/"neutral"
- `priority_level` TEXT — "high"/"medium"/"low"
- `cn_interpretation` TEXT — 中文投资解读（区别于机翻）

### 新增表
- `daily_briefs` — 每日简报缓存（日期、总结文本、更新时间）
- `sector_mentions` — 板块提及记录（日期、板块、账号、方向、推文ID）
